React useState Hook Guide

What is useState?

The useState hook allows you to add state to functional components in React. It is an alternative to useReducer for simple state updates.

Syntax:

const [state, setState] = useState(initialState)

// state: current value
// setState: function to update state
// initialState: initial value

How useState() Works

  1. Initialize State: Creates a state variable and updater function.
    const [count, setCount] = useState(0);
  2. State is Preserved Across Renders: React remembers the state value between re-renders.
  3. State Updates with Updater Function: Calling setCount(newValue) updates the state and triggers re-render.
  4. Triggers Re-render: Only the component using useState re-renders, keeping the UI updated.

1. Counter Example

import { useState } from 'react';

export default function Counter() {
    const [count, setCount] = useState(0);

    function handleClick() {
        setCount(count + 1);
    }

    return (
        <button onClick={handleClick}>
            Click {count} me
        </button>
    );
}

Explanation:

  • useState(0): initializes count with 0.
  • setCount(count + 1): increments the value.

2. Managing Form Input State

import React, { useState } from 'react';

function Form() {
    const [name, setName] = useState('');
    const [age, setAge] = useState('');
    const [submitted, setSubmitted] = useState(false);

    const handleSubmit = () => setSubmitted(true);

    return (
        <div>
            <input
                type="text"
                value={name}
                onChange={(e) => setName(e.target.value)}
                placeholder="Enter your name"
            />
            <input
                type="number"
                value={age}
                onChange={(e) => setAge(e.target.value)}
                placeholder="Enter your age"
            />
            <button onClick={handleSubmit}>Submit</button>
            {submitted && <p>Form Submitted!</p>}
        </div>
    );
}

export default Form;

Explanation:

  • useState(''): initializes name and age with empty strings.
  • onChange updates the state as the user types.
  • setSubmitted(true): marks the form as submitted.

Interesting Facts About useState

  • Manages simple state like counters, toggles, and form inputs.
  • Functional updates for previous state: setCount(prev => prev + 1)
  • State updates are asynchronous.
  • State does not merge automatically; you need to spread objects manually.
  • Can hold any data type: numbers, strings, objects, arrays, or functions.
  • Lazy initialization: useState(() => computeInitialValue())
  • Re-renders are skipped if state value is the same.

When to Use useState

  • For simple state management in functional components.
  • When state changes over time.
  • When no complex updates or dependencies are required.
