React.js Folder Structure

Interactive guide with all default files and detailed explanations

  • my-app/
    • node_modules/

      Contains all npm packages and dependencies installed. Auto-generated.

    • public/
      • index.html

        Main HTML template. React mounts the app at <div id="root">.

      • favicon.ico

        Browser tab icon.

      • manifest.json

        Config for Progressive Web App (PWA).

      • robots.txt

        Instructions for web crawlers.

      • assets/

        Static files like images, icons, fonts.

    • src/
      • assets/

        Images, icons, and fonts used in the app.

      • components/
        • Button/
          • Button.js - React component code.
          • Button.css - Component-specific styles.
          • Button.test.js - Unit tests for Button.
      • pages/
        • Home.js - Home page component.
        • About.js - About page component.
        • Contact.js - Contact page component.
      • services/
        • api.js - Axios/fetch setup.
        • auth.js - Auth API calls.
        • user.js - User-related API calls.
      • hooks/
        • useAuth.js - Custom hook for authentication.
        • useFetch.js - Custom fetch hook.
        • useLocalStorage.js - Custom hook for local storage.
      • contexts/
        • AuthContext.js - Global auth state.
        • ThemeContext.js - Global theme state.
      • utils/
        • validators.js - Validation helper functions.
        • formatDate.js - Date formatting helpers.
        • constants.js - App constants.
      • App.js - Main component wrapping the app.
      • App.css - Styles for App.js.
      • index.js - Entry point, renders App to root.
      • index.css - Global CSS styles.
      • logo.svg - React logo (default).
      • reportWebVitals.js - Performance measuring utility.
      • setupTests.js - Jest and React Testing Library setup.
    • .gitignore - Git ignore file.
    • package.json - Project metadata, dependencies, scripts.
    • package-lock.json / yarn.lock - Locks package versions.
    • README.md - Project documentation.
    • .env - Environment variables.
