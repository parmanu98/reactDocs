React.js Commands

Common commands with detailed explanations

Create React App
npx create-react-app my-app

Creates a new React project called my-app with all default setup including Webpack, Babel, React scripts, and folder structure. 

yarn create react-app my-app

Alternative using yarn to create a new React app.

Start Development Server
npm start

Starts the React development server at http://localhost:3000. Supports hot reloading – changes in your code automatically update the browser. 

yarn start

Alternative command using Yarn.

Build Project
npm run build

Builds the app for production in the build/ folder. Optimizes files, minifies JS/CSS, and prepares the project for deployment. 

yarn build

Alternative using Yarn.

Run Tests
npm test

Starts Jest test runner in interactive watch mode. Detects changes in test files automatically and reruns tests. 

yarn test

Alternative using Yarn.

Lint & Fix Code
npm run lint

Runs ESLint to check for coding standard violations. 

npm run lint -- --fix

Automatically fixes fixable linting issues.

Install Dependencies
npm install package-name

Installs a package and updates package.json.

yarn add package-name

Yarn equivalent for installing a package.

Remove Dependencies
npm uninstall package-name

Removes a package and updates package.json.

yarn remove package-name

Yarn equivalent.

Serve Production Build
npx serve -s build

Serve the production build locally. Must have serve installed globally or via npx.

Other Useful Commands
  • npm start or yarn start – Starts dev server.
  • npm run build or yarn build – Builds project for production.
  • npm test or yarn test – Run tests.
  • npm install package-name – Install package.
  • npm uninstall package-name – Remove package.
  • npm run eject – Eject CRA configuration (advanced, irreversible).
  • npm run lint – Run ESLint.
  • npx create-react-app my-app – Create new React app.
