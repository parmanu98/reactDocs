React.js Commands

Here are the most common React commands for development, building, testing, and managing dependencies.

Create React App
npx create-react-app my-app

Creates a new React app with default configuration.

yarn create react-app my-app

Yarn alternative.

Start Development Server
npm start
yarn start

Starts the development server with hot reload at http://localhost:3000.

Build Project
npm run build
yarn build

Generates an optimized production build in the build/ folder.

Run Tests
npm test
yarn test

Starts Jest test runner in watch mode.

Install / Remove Packages
npm install package-name
yarn add package-name
npm uninstall package-name
yarn remove package-name

Install or remove npm/yarn packages.

Linting
npm run lint
npm run lint -- --fix

Check code style and optionally auto-fix issues.

Serve Production Build
npx serve -s build

Serve the production build locally.

Eject CRA
npm run eject

Expose configuration files for advanced customization. Irreversible.
