React Router

React Router allows navigation between different pages/components in a React app.

Basic Routing

Setup using BrowserRouter and Route components.

import { BrowserRouter, Routes, Route } from 'react-router-dom';
import Home from './pages/Home';
import About from './pages/About';

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Home />} />
        <Route path="/about" element={<About />} />
      </Routes>
    </BrowserRouter>
  );
}
Link and NavLink

Use Link or NavLink to navigate without page reload.

import { Link, NavLink } from 'react-router-dom';

function Navbar() {
  return (
    <nav>
      <Link to="/">Home</Link>
      <NavLink to="/about">About</NavLink>
    </nav>
  );
}
Nested & Dynamic Routes
<Route path="/users">
  <Route path=":id" element={<UserDetail />} />
</Route>

Dynamic route example: /users/123

404 Page
<Route path="*" element={<NotFound />} />

Catch-all route for pages that do not exist.
