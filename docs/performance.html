Performance Optimization in React

Learn how to optimize React applications for speed and responsiveness.

React.memo
const Expensive = React.memo(({ value }) => {
  console.log('Rendering', value);
  return <div>{value}</div>;
});

Prevents re-rendering if props haven't changed.

useMemo
import { useMemo } from 'react';

const result = useMemo(() => expensiveCalculation(data), [data]);

Memoizes values to avoid expensive recalculations.

useCallback
const handleClick = useCallback(() => { console.log('clicked'); }, []);

Memoizes functions to prevent unnecessary re-renders of children.

Lazy Loading & Suspense
import React, { lazy, Suspense } from 'react';
const Component = lazy(() => import('./Component'));

<Suspense fallback=<div>Loading...</div>>
  <Component />
</Suspense>

Load components only when needed to reduce initial bundle size.
