Styling in React

React supports multiple ways to style components.

CSS Files
/* App.css */
.app { background-color: #f0f0f0; }

Import CSS directly into components using import './App.css'.

CSS Modules
/* Button.module.css */
.button { background: blue; color: white; }

import styles from './Button.module.css';
<button className={styles.button}>Click</button>
Styled Components
import styled from 'styled-components';

const Button = styled.button`
  background: green;
  color: white;
  padding: 0.5rem 1rem;
`;

<Button>Click</Button>
Tailwind CSS
<button className="bg-indigo-600 text-white px-4 py-2 rounded">Click</button>

Utility-first CSS for rapid styling.
