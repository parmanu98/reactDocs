React Folder Structure

Here is the standard folder structure of a React project created with create-react-app, including default files and directories.

Root Folder: my-app/
  • node_modules/ – Contains all npm dependencies.
  • public/ – Static files like index.html, favicon, manifest.
  • src/ – Application source code.
  • .gitignore – Files ignored by Git.
  • package.json – Project metadata and dependencies.
  • package-lock.json / yarn.lock – Exact dependency versions.
  • README.md – Project documentation.
  • .env – Environment variables.
public/ Folder
  • index.html – Main HTML template.
  • favicon.ico – Browser tab icon.
  • manifest.json – PWA configuration.
  • robots.txt – Search engine crawler instructions.
  • assets/ – Images, fonts, icons.
src/ Folder
  • index.js – Entry point for React app.
  • App.js – Main component.
  • App.css – Styles for App component.
  • index.css – Global styles.
  • logo.svg – React logo.
  • reportWebVitals.js – Performance metrics.
  • setupTests.js – Test setup for Jest.
  • components/ – Reusable UI components.
  • pages/ – Route-level page components.
  • hooks/ – Custom hooks.
  • contexts/ – React Context providers.
  • services/ – API calls and services.
  • utils/ – Utility functions and constants.
