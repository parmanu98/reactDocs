React Basics

This page covers the foundational concepts of React, including JSX, Props, State, Event Handling, Conditional Rendering, and Forms.

JSX

JSX allows you to write HTML-like code inside JavaScript. Key points:

  • Use curly braces {} for expressions.
  • Use className instead of class.
  • Elements must be wrapped in a single parent element.
function App() {
  const name = "React";
  return (
    <div>
      <h1>Hello, {name}!</h1>
    </div>
  );
}
Props

Props allow passing data from parent to child components.

function Greeting(props) {
  return <h2>Hello, {props.name}!</h2>;
}

function App() {
  return <Greeting name="Alice" />;
}
State

State allows a component to manage internal data.

import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}
Event Handling

React events are handled using camelCase syntax:

function Button() {
  function handleClick() {
    alert("Button clicked!");
  }

  return <button onClick={handleClick}>Click Me</button>;
}
Conditional Rendering

You can render elements conditionally using JavaScript operators.

function LoginMessage({ isLoggedIn }) {
  return (
    <div>
      {isLoggedIn ? <p>Welcome back!</p> : <p>Please log in.</p>}
    </div>
  );
}
Forms

Controlled components handle form input via React state.

import { useState } from 'react';

function NameForm() {
  const [name, setName] = useState("");

  return (
    <form onSubmit={(e) => { e.preventDefault(); alert(name); }}>
      <input value={name} onChange={(e) => setName(e.target.value)} />
      <button type="submit">Submit</button>
    </form>
  );
}

Uncontrolled components use ref to access DOM values directly.
